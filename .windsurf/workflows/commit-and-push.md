---
description: Commit and push changes to git
---

This workflow commits and pushes changes to git.

1. Run `git add .` to stage all changes
2. Run `git commit -m "<your commit message>"` to commit changes
3. Run `git push` to push changes to remote

Usage: Use this workflow when you want to commit and push your changes to git. Replace `<your commit message>` with a descriptive commit message.
